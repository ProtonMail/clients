# Documentation 

This directory contains a collection of documentation articles as well as RFC.

* [**general**](./general): General purpose documentation, useful information
* [**guides**](./guides): Guides/best practices for tools, libraries, frameworks, etc...
* [**specifications**](./specifications): Technical documentation on how things work
* [**rfc**](./rfcs): RFC collection

# Note
